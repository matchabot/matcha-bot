# Getting started

Install MatchaBot and run your first generator.

## How to start

### 1 - Add matchabot to your project

> Pre-requisites: you have a in a directory that contains a `package.json` file.

Execute the following command at the root of your project.

```bash
yarn add matchabot --dev
```

or

```bash
npm install matchabot --save-dev
```

**Matchabot** can be installed **globally** with

```bash
npm install -g matchabot
```

### 2ï¸ - Initialise matchabot

Execute the following command at the root of your project.

```bash
matchabot init
```
